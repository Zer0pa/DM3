# DM3 2026 Restart: Geometry-First Computational Manifold Falsification

## What This Is

This project is a restart of DM3 as a falsifiable, source-backed research program. It treats the geometry-first manifold manifesto as a seed document and claims inventory, then rebuilds only from recoverable code, reproducible tests, and cross-platform receipts rather than inherited narrative.

The immediate deliverable is not a new model. It is a rigorous restart package: a mapped test registry, a source-backed deterministic baseline, a RedMagic 10 Pro execution lane, and a decision on which claims survive re-testing.

## Core Research Question

Can any source-backed DM3 geometry-first substrate be reconstructed and falsified from recoverable evidence only, then reproduced across Mac host and RM10 Pro execution lanes without relying on unrecovered hybrid source?
