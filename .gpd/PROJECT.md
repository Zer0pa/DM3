# DM3 2026 Restart: Geometry-First Computational Manifold Falsification

## What This Is

This project is a restart of DM3 as a falsifiable, source-backed research program. It treats the geometry-first manifold manifesto as a seed document and claims inventory, then rebuilds only from recoverable code, reproducible tests, and cross-platform receipts rather than inherited narrative.

The 3D Double Meru is the primary scientific object of inquiry. The computational field-computer reading is the leading live hypothesis about what that object may be. The later hybrid DM3 residue is evidence about that object. The older Genesis / governed deterministic layers remain the witness floor, comparison surface, and anti-delusion machinery.

The immediate deliverable is not a new model. It is a rigorous scientific reconstruction package: a Double Meru geometry dossier, a fidelity-classified hybrid archaeology record, a preserved source-backed witness baseline, a RedMagic 10 Pro execution lane, and a decision on which claims survive re-testing.
