# Notation Glossary

## Core Terms

- `Txx`: numbered manifesto claim or test label from `docs/reference/GEOMETRY_FIRST_MANIFESTO.md`
- `boundary`: retained outer node set for readout or forcing
- `bulk`: interior node set reconstructed or evolved under the manifold rules
- `rho(J)`: contraction or local Jacobian spectral-radius witness
- `E`: energy or Lyapunov-like witness, if source-backed
- `RM10 Pro`: RedMagic 10 Pro device lane
- `receipt`: machine-readable execution artifact proving what actually ran
- `ledger`: ordered collection of receipts, hashes, or summaries used for replay comparison
