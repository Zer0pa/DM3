## Out Of Scope

- treating interesting behavior as a success proxy

## Progress That Should Not Count As Success

- restoring demos without the authority metric
- porting code without resolving the scientific claim
- adding GPU paths before the baseline substrate and acceptance gates are stable

## Rethink Triggers

- if the manifold cannot support a crisp encoding-and-readout story
- if the geometry does not improve the target computation relative to simpler baselines
- if the acceptance metric collapses into aesthetic or narrative criteria
