# DM3 FAQ

## What is DM3?

DM3 is a graph-structure diagnostic built around a compiled binary and a fixed
380-vertex C3-symmetric graph.

## Why is there a Python package in this repo?

The package is a thin review utility so `pip install .` works from the clone
root and CI can validate the repo surface consistently.

## Is the root README the live engineering source of truth?

No. The root README is the canonical review surface. Live session work stays in
`docs/restart/` and `artifacts/` until promoted.

## Does DM3 inherit novelty claims from another ZPE product?

No. DM3 stands on its own and needs its own product-specific novelty schedule.
