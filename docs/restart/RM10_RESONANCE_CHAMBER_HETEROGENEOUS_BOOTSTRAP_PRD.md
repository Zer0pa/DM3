1. Can the attached RM10 be treated as a coupled computational environment in

4. What one next move is justified if heterogeneity-first probing is taken
   seriously?

This phase will use the smallest anchor family current surfaces can expose:

- `delta_E`
- `coherence`

Abstain rule:

- lanes are not comparable
- NPU remains non-callable

## Resonance-Chamber Meaning

For this phase, "resonance chamber" means:

- repeated forcing through a stateful environment
- lane-specific operators acting on the same evolving state
- trajectory behavior that may show thresholds, persistence, or collapse
- a coupled system whose scientific content is in the stability class or
  attractor class, not just one end-state hash

It does not mean literal proof of Kuramoto synchronization, holography, or
field ontology.
