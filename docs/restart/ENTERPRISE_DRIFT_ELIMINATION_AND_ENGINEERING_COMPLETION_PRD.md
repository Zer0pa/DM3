# Enterprise Drift Elimination And Engineering Completion PRD

Last refreshed: `2026-04-05`

## Mission

Turn the retained `01.2.3.4.1` package into a clean, truthful, branch-local
engineering control surface for the next operator.

This is not a proof campaign.
This is not a heterogeneous-reopen campaign.
This is not a science-story expansion.

The purpose is narrower:

- stop startup drift
- preserve authority
- keep the RM10 instrument honest
- close the remaining same-family execution boundary
- return one exact next move without widening the claim ceiling

## Operating Posture

Keep this order fixed:

1. engineering truth
2. receipts and telemetry
3. bounded interpretation

Do not invert it.

The Red Magic 10 Pro remains the primary branch-local engineering instrument.
The Mac remains the only live `source_built` authority lane.

An abstain is acceptable.
A negative result is acceptable.
A hard boundary is acceptable.
Uncontrolled drift is not acceptable.
