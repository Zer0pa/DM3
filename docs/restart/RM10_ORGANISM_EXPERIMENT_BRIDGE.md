## Battery Family After Confirmation

### 1. Homeostasis Battery

Question:
Does the same-family packet settle to the same response class repeatedly under
the same envelope?

Track:

- `delta_E`
- `coherence`
- completion class
- receipt integrity
- run-to-run variance

### 2. Lesion / Repair Battery

Question:
What happens if one boundary condition or asset is perturbed and then restored?

Track:

- failure mode
- time or windows to recovery
- whether the previous tuple reappears after restoration

### 3. Transplant / Resume Battery

Question:
Can a retained intermediate state be resumed or moved without destroying the
response class?

Track:

- state carry-over success
- response drift after resume
- whether history matters more than final inputs

### 4. Morphological Contribution Battery

Question:
How much of the behavior is carried by morphology and environment versus the
explicit controller path?

Track:

- sensitivity to row order
- sensitivity to persistence
- sensitivity to boundary state
- effect of stripped-down versus richer controller scaffolds

### 5. Boundary Adapter Battery

Question:
Only after a stable same-family surface exists, can a thin transformer-style
adapter improve interfacing without taking over the hidden-state substrate?

Rule:

- keep it thin
- keep it boundary-local
- compare it against the same geometry-first metrics

## Measurement Map

| Organism-style term | Practical measurement |
| ------------------- | --------------------- |
| homeostasis | repeated-run variance under identical envelope |
| regeneration | recovery after reversible perturbation |
| memory | dependence on recent window history |
| lesion sensitivity | tuple change after boundary or asset damage |
| morphological contribution | performance delta when morphology/environment changes but controller intent stays fixed |

## Stop Rules

- Stop if the confirmation replay is not reproducible.
- Stop if the effect disappears when the environment is reset cleanly.
- Stop if the only apparent signal is runtime or heat.
- Stop if a boundary adapter becomes the real hidden-state substrate.

holds.
