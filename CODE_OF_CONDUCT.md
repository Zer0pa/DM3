# Code of Conduct

## Standard

Work in this repository is expected to be direct, evidence-led, and respectful.
Disagreement is normal; distortion, harassment, and sabotage are not.

## Expected Behavior

- challenge claims with evidence
- keep criticism technical and specific
- preserve the audit trail
- respect the operator, reviewers, and other agents' boundaries

## Unacceptable Behavior

- harassment or personal attacks
- evidence deletion or concealment
- knowingly misrepresenting results, scope, or authorship
- pushing others to accept a narrative over the governing metric

## Enforcement

The repository owner may remove comments, close contributions, or restrict
access when behavior conflicts with these standards.
