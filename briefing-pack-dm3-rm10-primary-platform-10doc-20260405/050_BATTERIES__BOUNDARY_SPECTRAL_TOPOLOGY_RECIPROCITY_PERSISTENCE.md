# Batteries

## Global Entry Rule

Boundary, spectral, topology, reciprocity, and persistence are scientific
batteries, not theme buckets. Each battery must:

- inherit the same receipt-backed RM10 CPU control observable
- include at least one matched null or comparator row
- remain `ABSTAIN` if the observable family changes or loses receipt comparability

## Minimum Rows Before Promotion

| Family | Minimum rows before anything stronger than `ABSTAIN` | Main null or comparator |
| --- | --- | --- |
| boundary | `B-01`..`B-03` | size-matched interior and random masks |
| spectral | `S-01`..`S-04` | low-order-summary and order-destruction nulls |
| topology | `T-01`..`T-03` | surrogate null preserving scalar marginals |
| reciprocity | `R-01`..`R-03` | arbitrary relabeling null |
| persistence | `P-01`..`P-05` | hard-reset washout and thermal or runtime drift nulls |
