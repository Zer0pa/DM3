# Double Meru Object And Surviving Claims

## Purpose

This document merges the current science-side bottom line from the geometry and
hybrid-fidelity work so the team can reason from one compact object-level
surface.

## Strongest Recovered Object

The strongest source-backed read is still:

`a mirrored dual 3D geometry with a privileged waist / throat layer, wired for deterministic settling and modal tests`

What is recovered as fact:

- the surviving source builds a mirrored dual-volume structure rather than a
  decorative shell
- the geometry has a concrete privileged waist / throat at the deduplicated
  `z = 0` layer
- the lift is deterministic and rational, not a floating-point art asset
- settling and resonance code operate directly on the geometry
- plus-cone, minus-cone, throat-mode, reciprocity, orthogonality, and
  crosstalk observables are real source-backed surfaces

## Strongest Hybrid Read

The strongest surviving hybrid story is still:

`boundary-oriented readout + nodewise micro-transformer + scar-memory augmentation on top of a geometry-first substrate`

What survives:

- boundary-oriented readout is the strongest surviving hybrid motif
- nodewise local intelligence remains more justified than a pure global adapter
- scar-memory survives as a real augmentation line
- the late compiled hybrid line is mixed but substantively geometry-coupled

What does not currently survive:

- a recovered dedicated center-model architecture
- a governed non-stub hybrid task
- a governed GPU lane
- a claim that the late hybrid definitely deepened the geometry rather than
  partly obscuring it

## Smallest True Scientific Statement

The Double Meru is still best treated as a geometry-first computational body,
not a narratively reconstructed modern ML system.

The strongest narrow claim today is:

`the source-backed Double Meru defines a deterministic dual-volume geometry with a privileged throat and computation-oriented settling / modal structure, while the later hybrid residue preserves some boundary and nodewise continuation but does not currently recover a living same-binary G2 route`

## What Remains Open

  generation
- which parts of the richer residue family can still be linked to a callable
  adjacent launcher surface

  redevelopment

Engineering should treat the hybrid residue as evidence about that object, not
as recovered authority.
